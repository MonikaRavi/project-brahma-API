API

/v1/AX2009/salesOrderList

get sales order list from AX2
JSON Structure
{ "SalesId": Sales Id, "Amount":amount , "createdDate": createdDate, "CUSTACCOUNT":customer account, "Customer":Customer name, "SalesName":Sales Name, "RecentPickUp":recent pickup date }

/v1/AX2009/customerDetailsFromSalesId/:SalesId

Give the customer Details of AX2009 from SalesID
:salesId for trial {SO-1787116}
JSON Structure
[ { "SalesId": Sales Id, "CustAccount": Customer Account, "customer":Customer Name, "Phone":Phone Number "Email": Email, "Address": Address, "RSD": } ]

/v1/AX2009/salesOrderDetailsFromSalesId/:salesId

Get sales order details from sales Id for AX2009
:salesOrder for trial {SO-1787116}
JSON Structure
[ { "SalesId": Sales Id, "SalesName": Sales Name, "createdDate": createdDate, "customer":Customer Name, "ItemID": Item ID, "ItemName":Name of Item, "SalesQty": Quantity sold, "LineAmount": Amount, "SalesStatus": Status of sale }, { ... }, ... ]

/v1/AX2009/customerSalesList/:AxCustomerId

returns customer Sales List by AX Account Number
:AxCustomerId for trial {00764}
JSON Structure
{ "CustAccount":customer account, "Customer":customer Name }, { "SalesId": Account Name, "createdDate": Probability%, "amount": Amount }

/v1/salesforce/salesOrderDetailsFromSalesId/:salesId

Get sales order details from sales Id for salesforce
:salesOrder for trial {SO-1787116}
JSON Structure
[ { "Name": sales Name, "qAmount": line Amount, "so": Sales Order Number, "erpAmount":ERP Final Amount, "accountID": account ID, "closeDate":close Date, "deliveryState":Delivery state }, ]

/v1/salesforce/opportunities/:SFCustomerId

returns Opportunity by salesforce Account Number
:AxCustomerId for trial {00764}

{ "name": Account Name, "probability": Probability%, "amount": Amount }

/v1/salesforce/distributor/:type

returns Distributor Detail by Distributor Type
:type can be {safety,plumbing}
JSON Structure
{ "accountId": AccountID, "Name": Account Name, "Type": Distributor Type, "Phone": Contact Number, "Website": Distributor website }
